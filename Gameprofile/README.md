# GameStore Demo

A minimal static demo that resembles a Steam-like game store.

How to run:

Open `index.html` in your browser.

Files:
- `index.html` — main page
- `styles.css` — styles
- `app.js` — client-side logic and sample data
- `covers/` — downloaded Steam header cover images for 30 games

Notes:
- This is a static demo with localStorage-based cart and sample data.
- The site now uses downloaded Steam-style cover images for 30 popular games.
- Extend by adding purchase flows, store APIs, or more titles.
